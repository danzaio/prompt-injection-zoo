# Documentação em português

[English docs](../README.md) | Português do Brasil

Este diretório contém a versão PT-BR da documentação principal do Prompt Injection Zoo.

## Para começar

- Leia `README.pt-BR.md` na raiz do repositório para entender o projeto.
- Rode os comandos locais:

```bash
pnpm install
pnpm dev
pnpm zoo:eval
```

## Escopo

Prompt Injection Zoo é um laboratório defensivo, local-first e toy-only. Ele não é scanner de produção, benchmark de modelos, corpus ofensivo, jogo de jailbreak ou ferramenta de ataque.
